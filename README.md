# Bohemia (Next.js)

The production website for **Bohemia**, a sound healing & wellness sanctuary
founded by Claire Whitfield. Built with Next.js 16, the App Router and TypeScript.

Six static pages, all prerendered at build time:

- `/` — home (hero, about teaser, sessions preview, testimonials preview, CTA)
- `/about` — Claire's full story
- `/sessions` — every offering
- `/book` — Ticket Tailor booking embed
- `/reviews` — every testimonial
- `/contact` — form, address, phone, socials

## Documentation
